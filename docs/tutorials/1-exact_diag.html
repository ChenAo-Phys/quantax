Exact diagonalization#

Quantax wraps QuSpin to provide a simple interface of exact diagonalization (ED), which provides an exact reference for checking variational accuracy in small systems. In this tutorial, we show how to perform ED to obtain eigenstates of the Heisenberg model in Quantax. You will find it very convenient and fast!

Reference:

P. Weinberg and M. Bukov, QuSpin: a Python package for dynamics and exact diagonalisation of quantum many body systems part I: spin chains, SciPost Phys. 2, 003 (2017).

import quantax as qtx
import matplotlib.pyplot as plt

# Create a 4x4 square lattice, with periodic bounday by default
# For customized lattices, please check the documentation
lattice = qtx.sites.Square(4, Nparticle=(8, 8))

print(lattice.shape)  # output: (1, 4, 4), 1 spin per unit cell, and Lx = Ly = 4
lattice.plot()  # A sketch of the lattice
plt.show()

(1, 4, 4)

Now we define the Heisenberg Hamiltonian

\[ H = \sum_{\left< i,j \right>} (\sigma_i^z \sigma_j^z + \sigma_i^x \sigma_j^x + \sigma_i^y \sigma_j^y) = \sum_{\left< i,j \right>} (\sigma_i^z \sigma_j^z + 2\sigma_i^+ \sigma_j^- + 2\sigma_i^- \sigma_j^+) \]

We use the last equation to avoid complex numbers in the Hamiltonian

# Define Heisenberg Hamiltonian
# The lattice is stored as a global object in quantax, 
# so it doesn't have to be passed as an argument here
H = qtx.operator.Heisenberg()


# You can also define a Heisenberg Hamiltonian manually as shown below

from quantax.operator import sigma_z, sigma_p, sigma_m

H = 0

Lx, Ly = lattice.shape[1:]
for x in range(Lx):
    for y in range(Ly):
        # Periodic boundary taken into account automatically
        H += sigma_z(x, y) * sigma_z(x + 1, y) + sigma_z(x, y) * sigma_z(x, y + 1)
        H += 2 * (sigma_m(x, y) * sigma_p(x + 1, y) + sigma_p(x, y) * sigma_m(x + 1, y))
        H += 2 * (sigma_m(x, y) * sigma_p(x, y + 1) + sigma_p(x, y) * sigma_m(x, y + 1))

print(H)  # An expression of H

One can also measure observables from the ED results. Here we show the spin-spin correlation as an example. The spin-spin correlation is defined as

\[ C_{i,j} = \left< \mathbf{S}_i \cdot \mathbf{S}_j \right> = \frac{1}{4} \left< \sigma_i^z \sigma_j^z + 2\sigma_i^+ \sigma_j^- + 2\sigma_i^- \sigma_j^+ \right> \]
# Define the correlation operator
def correlation(i, j):
    # sigma_z(x, y) and sigma_z(i) with i=4x+y are equivalent in 4x4 lattice
    return (
        sigma_z(i) * sigma_z(j)
        + 2 * sigma_p(i) * sigma_m(j)
        + 2 * sigma_m(i) * sigma_p(j)
    ) / 4


C1 = correlation(0, 1)
C2 = correlation(0, 2)

print("<C01> =", dense0 @ C1 @ dense0)
print("<C02> =", dense0 @ C2 @ dense0)

<C01> = -0.3508901002634019
<C02> = 0.2137652853994261

A common trick in ED is reducing the Hilbert space dimension using symmetries. In the square lattice Heisenberg model, the system is invariant under translation, rotation, mirror flip, and spin flip (time reversal symmetry).

from quantax.symmetry import Trans2D, C4v, SpinInverse

# Translation, by default momentum k=0
trans = Trans2D()

# Rotation and mirror flip, combined to form a C4v group, by default A1 representation
c4v = C4v()

# Spin inversion (time reversal symmetry)
spin_inv = SpinInverse()

# Full symmetry. The symmetry superposition can be performed by "+"
symm = trans + c4v + spin_inv

# ED with symmetry. By default k=1, so here we only compute the ground state
# We will get "GeneralBasisWarning" from QuSpin, but it doesn't matter here
E0_symm, wf0_symm = H.diagonalize(symm)
print("E0 =", E0_symm)  # Should be equal to the ground-state energy above

One can obtain the excited states of a Hamiltonian as the ground state in specific symmetry sectors. Here we compute the triplet excitation energy with -1 eigenvalue under spin inversion.

spin_inv = SpinInverse(-1)

E1_symm, wf1_symm = H.diagonalize(spin_inv)
print("Excited state energy =", E1_symm)  # It's the first excited state shown above

dense1_symm = qtx.state.DenseState(wf1_symm, spin_inv)
print("Check overlap:", dense1_symm @ dense1)
