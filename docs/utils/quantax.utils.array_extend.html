quantax.utils.array_extend¶

quantax.utils.array_extend(array: Array, multiple_of_num: int, axis: int = 0, padding_values: Number = 0) → Array¶

Extend the array.

Parameters:

  • array – The array to be extended.

  • multiple_of_num – Specify the size of the extended axis to be a multiple of this number.

  • axis – The axis to be extended, default to 0 (the first dimension).

  • padding_values – The padding values, default to 0.
