quantax.operator.S_y#

quantax.operator.S_y(*index) → Operator#

\(S^y\) operator

\[\begin{split}S^y = \begin{pmatrix} 0 & -i/2 \\ i/2 & 0 \end{pmatrix}\end{split}\]

Warning

The \(S^y\) operator works only if the default data type is set to complex in quantax.set_default_dtype.
