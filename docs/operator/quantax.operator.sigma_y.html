quantax.operator.sigma_y¶

quantax.operator.sigma_y(*index) → Operator¶

\(\sigma^y\) operator for spin and fermion systems.

For spin systems,

\[\begin{split}\sigma^y = \begin{pmatrix} 0 & -i \\ i & 0 \end{pmatrix}\end{split}\]

Warning

The \(\sigma^y\) operator works only if the default data type is set to complex in quantax.set_default_dtype.
