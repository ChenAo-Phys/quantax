quantax.nn.ConvSymmetrize¶

class quantax.nn.ConvSymmetrize¶

Bases: NoGradLayer, RawInputLayer

Symmetrize the output of a convolutional network according to the given symmetry.

__init__(symm: Symmetry | None = None)¶
Parameters:

symm – The symmetry used for symmetrization, by default TransND with sectors 0. If Identity is given, the layer won’t symmetrize its output.
